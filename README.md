# 🛒 Supermarket Sales Analysis Dashboard (Power BI)

## 📌 Project Overview

This project analyzes supermarket sales data from three branches (A, B, and C) between **January and March 2019** using **Power BI**. The objective is to transform raw sales data into meaningful business insights through interactive visualizations and dashboards.

The dashboard helps compare branch performance, identify top-selling product categories, understand customer purchasing behavior, and monitor sales trends over time.

## 🎯 Objectives

* Analyze overall sales performance and product demand.
* Compare sales across different branches.
* Evaluate customer satisfaction using product ratings.
* Identify sales trends by month and day type.
* Build an interactive Power BI dashboard for business decision-making.

## 📊 Dashboard Features

* **Total Sales, Transactions, and Gross Income KPIs**
* **Branch-wise and Product Line Sales Analysis**
* **Average Customer Rating by Product Line**
* **Monthly Sales Trend by Branch**
* **Weekday vs Weekend Sales Distribution**
* **Quantity Sold by Product Category**
* Interactive **slicers** for Branch and Product Line filtering

## 🛠 Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Cleaning & Transformation
* Data Visualization

## 📂 Dataset Information

* Source: Kaggle Supermarket Sales Dataset
* Records: 1,000 Transactions
* Period: January – March 2019
* Branches: A, B, C

## 🔍 Key Insights

* Total Sales generated: **322.97K**
* Total Transactions: **1,000**
* **Branch C** recorded the highest overall sales.
* **Electronic Accessories** had the highest quantity sold.
* Approximately **68.8%** of sales occurred on weekdays.
* **Food & Beverages** received the highest average customer rating.
* Sales declined in February across all branches and recovered in March.
